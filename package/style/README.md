## 介绍
用于xs组件库样式，实现xs不同版本之间共享，同样会提供类似于bootstrap类似的样式类。   

## 安装
> npm i -D @xunserver/xs-style

或者使用@xunserver/cli `xs add style`安装

## 使用
有两种安装方式
- 自行编译安装，支持修改变量。
- 直接引入css文件

除此之外也支持全局导入和局部导入
